Please run the python code using the 64 bit version of Python 3.12 to ensure full compatibility as this was the version used for development.
Install 64 bit version of Python 3 from https://www.python.org. 

Please install the following pre-requisites
pip3 install -U scikit-learn
pip3 install -U seaborn
pip3 install -U pandas
pip3 install -U numpy
pip3 install -U nltk
pip3 install -U requests

run script by navigating to this directory and running python "Coursework 1 - Part 2.py"
